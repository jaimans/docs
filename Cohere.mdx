title: 'Cohere'
description: 'Integrate with Cohere using OpenSesame'
---

Step 1. Install the OpenSesame SDK:

<CodeGroup>

```bash python 
pip install opensesame-sdk
```

</CodeGroup>

Step 2. Set up the OpenSesame_cohere client and add API keys

<CodeGroup>

```python 

from opensesame import OpenSesame_cohere

co = OpenSesame_cohere({
       'api_key': 'your_cohere_api_key',
       'open_sesame_key': 'your_opensesame_api_key',
       'project_name': 'your_opensesame_project_name',
   })

```
</CodeGroup>

Step 3. Make a request just like the Cohere sdk

<CodeGroup>
```python 

response = co.chat(
       preamble = "your system prompt",
       message = “user prompt”
   )

print(response.text)

```

</CodeGroup>
