Step 1. Install the OpenSesame SDK:

<CodeGroup>

```bash python 
pip install opensesame-sdk
```
```bash Javascript 
npm i opensesame-gemini
```

</CodeGroup>

Step 2. Set up the OpenSesame_gemini client and add API keys

<CodeGroup>

```bash python 

from opensesame import OpenSesame_gemini

client = OpenSesame_gemini({
    ‘api_key’ : ‘your_google_AI_api_key’,
    ‘open_sesame_key’ : ‘your_open_sesame_key’,
    ‘project_name’ : ‘open_sesame_project_name’ # Ensure this is correct
    })

```
```bash Javascript 

const OpenSesame = require('opensesame-gemini');

// Replace this code (your Gemini client)
const genAI = new GoogleGenerativeAI(process.env.API_KEY);

// with this :
const opensesame = new OpenSesame({
      apiKey: 'your Google API key here',
      openSesameKey: 'your OpenSesame key',
      projectName: 'your desired project name'
    });

```

</CodeGroup>

Step 3. Make a request just like the Gemini sdk

<CodeGroup>
```bash python 

model = client.GenerativeModel(model_name = “gemini-1.5-pro”)
response = model.generate_content(prompt = “User prompt”)


print(response.text)

```


</CodeGroup>
