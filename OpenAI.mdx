Step 1. Install the OpenSesame SDK:

<CodeGroup>

```bash python 
pip install opensesame-sdk
```

</CodeGroup>

Step 2. Set up the OpenSesame_openai client and add API keys

<CodeGroup>

```python 

from opensesame import OpenSesame_openai

client = OpenSesame_openai({
    ‘api_key’ : ‘your_openai_api_key’,
    ‘open_sesame_key’ : ‘your_open_sesame_key’,
    ‘project_name’ : ‘open_sesame_project_name’ # Ensure this is correct
    })
```

</CodeGroup>

Step 3. Make a request just like the OpenAI sdk

<CodeGroup>

```python 

response = client.chat.completions.create(
    model = “gpt-3.5-turbo”,
    messages = [{“role” : “user”, “content” : “user prompt”}]
)

print(response[‘choices’][0][‘message’][‘content])

```

</CodeGroup>
