Step 1. Install the OpenSesame SDK:

<CodeGroup>

```bash python 
pip install opensesame-sdk
```

</CodeGroup>

Step 2. Set up the OpenSesame_groq client and add API keys

<CodeGroup>

```python 

from opensesame import OpenSesame_groq   

client = OpenSesame_groq({
   'api_key': 'your_groq_api_key',
   'open_sesame_key': 'your_open_sesame_api_key',
   'project_name': 'your_open_sesame_project_name' # Ensure this is correct
   })


```
</CodeGroup>

Step 3. Make a request just like the Groq sdk

<CodeGroup>
```python 


response = client.ChatCompletions(client).create(
       model="llama3-8b-8192",
       messages=[
           {"role" : "system", "content" : "system prompt"},
           {"role": "user", "content": "User prompt ?"}
       ]
   )

print(response.choices[0].message.content)



```

</CodeGroup>
