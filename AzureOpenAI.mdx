---
title: 'Azure OpenAI'
description: 'Integrate with Azure OpenAI using OpenSesame'
---

Step 1. Install the OpenSesame SDK:

<CodeGroup>

```bash python 
pip install opensesame-sdk
```

</CodeGroup>

Step 2. Set up the OpenSesame_azure_openai client and add API keys, Azure endpoint, deployment name and api version

<CodeGroup>

```python 

from opensesame import OpenSesame_azure_openai

   client = OpenSesame_azure_openai({
        'api_key': 'Your Azure OpenAI api key',
        'azure_endpoint': 'You Azure endpoint',
        'deployment': 'Your Deployment name',
        'api_version' : '2024-06-01', # Enter the correct azure api version
        'open_sesame_key': "Your OpenSesame Key",
        'project_name': "your project name" # ensure this is correct
    })

```

</CodeGroup>

Step 3. Make a request to Azure OpenAI

<CodeGroup>

```python 

    response = client.call(
        prompt='user prompt',
        temperature=0.7,
        max_tokens=100
    )

    print(response)

```

</CodeGroup>
