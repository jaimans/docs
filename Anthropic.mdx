---

Step 1. Install the OpenSesame SDK:

<CodeGroup>

```bash python 
pip install opensesame-sdk
```

</CodeGroup>

Step 2. Set up the OpenSesame_anthropic client and add API keys

<CodeGroup>

```python 

from opensesame import OpenSesame_anthropic

client = OpenSesame_anthropic({
   'api_key': 'your_anthropic_api_key',
   'open_sesame_key': 'your_open_sesame_api_key',
   'project_name': 'your_open_sesame_project_name' # Ensure this is correct
   })

```
</CodeGroup>

Step 3. Make a request just like the Anthropic sdk

<CodeGroup>
```python 


response = client.Messages(client).create(
       model="claude-3-5-sonnet-20240620",
       max_tokens=1024,
       system="Your system prompt",
       messages=[
           {"role": "user", "content": "User prompt"}
       ]
   )


print(response[“content”][0][“text”])


```

</CodeGroup>
